# l10x Post-run: lets-opportunity-discovery

## Summarize the opportunity portfolio

After the `l10x research prd groom --modes opportunity_discovery` run completes, summarize the ranked portfolio:

```bash
python scripts/summarize-opportunity-portfolio.py
```

Check artifact completeness before summarizing:

```bash
python scripts/check-artifact-completeness.py
```

Review the portfolio using [`references/opportunity-review-checklist.md`](references/opportunity-review-checklist.md) before promoting any opportunity to PRD work.
