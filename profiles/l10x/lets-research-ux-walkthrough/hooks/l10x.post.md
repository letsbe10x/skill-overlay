# l10x Post-run: lets-research-ux-walkthrough

## Persona simulation pass (optional)

After completing the walkthrough, optionally run persona simulation on key screens to quantify which segments are most affected:

```bash
l10x persona simulate --repo-root . \
  --kind url --uri "https://example.com/signup" \
  --persona-id skeptical_buyer --scenario-id trust_and_click \
  --packs-root "$PACKS_ROOT" \
  --dry-run
```

Use the same cohort spec across screens to keep comparisons consistent.
