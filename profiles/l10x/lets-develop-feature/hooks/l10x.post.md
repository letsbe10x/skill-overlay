# lets Post-run: lets-develop-feature

## Update `.lets` artifacts and release runtime run lock (if applicable)

On completion:

1. Ensure `.lets/runs/develop-feature/<run_id>/` artifacts are updated:
   - `verification-record.md` reflects the Stage 8 verdict
   - `handoff.md` is ready for `lets-verify-change`
   - `run-state.json` marks Stage 9 as completed (or blocked)

2. If this was executed inside an engine-managed run, release the workspace lock to unblock subsequent pipeline stages:

```bash
lets run release $RUN_ID
```

If the goal exits with code 4 (workspace locked), release the oldest lock first:

```bash
lets run list
lets run release $OLDEST_RUN_ID
```

Retain the `run_id` for downstream stages (`lets-verify-change`, `lets-review-code`).
