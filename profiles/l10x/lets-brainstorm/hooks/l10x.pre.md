# l10x Pre-flight: lets-brainstorm

## Load workflow context and governance check

```bash
l10x run list --format json | jq '.[0].run_dir // empty'
```

If a `run_dir` is found:
1. Read `workflow_context.json` — extract `must_preserve.non_negotiables`, `must_preserve.critical_paths`, and `defaults.governance_posture`. If absent, log a warning and continue.
2. Read `classification.json` — extract `governance_verdict`. If absent, log a warning and continue.
3. If `governance_verdict == "BLOCK"`: stop, print the block reason, do not proceed.

If no active run directory exists, skip this pre-flight and continue.

## Context status check

```bash
l10x context status --format json
```

If `readiness_score < 50`, warn the user that context is thin and findings may be incomplete.
